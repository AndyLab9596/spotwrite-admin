機能追加の説明

機能追加は、Spotwriteの3つの機能の表示と非表示を選べる画面です。

在籍掲示板・モチベーションチェック・メッセンジャー機能はデフォルトで表示になっていますが、
Spotwriteの使い方に合わせてこの画面から非表示にすることもできます。

01

在籍掲示板の表示/非表示を選べます

在籍掲示板機能の表示と非表示を選べます。
在籍掲示板を利用すると、各メンバーが自分の在籍ステータスをSpotwrite上で編集し、みんなに掲示板で公開することができます。

在籍掲示板の画面

02

モチベーションチェックの表示/非表示を選べます

モチベーションチェック機能の表示と非表示を選べます。
利用すると、メンバーがログイン時に選択する5段階のモチベーションを分析画面のモチベーションチェックで確認できるようになります。
非表示を選ぶと、メンバーがモチベーションを選ぶ画面も非表示になります。

モチベーションチェック画面

03

メッセンジャーの表示/非表示を選べます

メッセンジャー機能の表示と非表示を選べます。
利用すると、リアルタイムでチャットのようにやりとりができるメッセージ機能を使えます。
他のツールでチャットや連絡を行なっている場合は、非表示にすることができます。
