無料デモをお試し

Spotwriteには、社内を活性化させる様々な機能をご用意。
ポジティブな情報を社内に共有することで従業員のエンゲージメントを高め、利用状況を可視化してコミュニケーションを育てることができます。

無料デモをお試し

  • POINT 1

    エンタープライズ領域で利用されているサーバー「Amazon Web Service」を利用しています。

  • POINT 2

    ワークスメディアの関連会社であるアンドライフ社にて、ISO（国際標準化機構）とIEC（国際電気標準会議）が共同で定めた、情報セキュリティの管理に関する国際規格 ISMS認証を取得。
    社員は規格に定められた厳格な情報セキュリティの取り扱いルールを遵守し、情報・プライバシーの保護に努めます。
    ＞情報セキュリティ基本方針

  • POINT 3

    すべてのデータはSSL/TLS暗号化通信化。バックアップ体制もあります。
    管理画面や個人のプライバシー情報は従業員ごとに権限操作できるため、アクセスコントロール機能も充実しています。
